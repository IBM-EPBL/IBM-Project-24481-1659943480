About the Project

Handwritten Digit Recognition is the need-of-the-hour technology in the current scenario. This Digit Recognition System is used to identify the digits from different sources like invoices, posts, email and so on. Since people relied on entering the digit from the sources manually, this technology was required to be properly implemented. This could help in reducing the manual errors and the difficulty in efficiently storing and accessing physical data. This project performs the recognition of handwritten digits [0-9] from the MNIST dataset, using a Convolutional Neural Network.
